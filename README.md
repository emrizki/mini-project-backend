Membuat sebuah mini project menggunakan NodeJS / Go dengan kriteria berikut:

- Memiliki API login untuk autentikasi pengguna dan tidak bisa diakses oleh publik.
- Memiliki API untuk CRUD data dengan otorisasi yang didapat saat login.
- Memiliki API logout dengan otorisasi yang didapat saat login.
- CRUD menggunakan Database SQL atau NoSQL.
- Setiap API menggunakan Request dan Response dalam format JSON.
- Setiap API menggunakan kode status yang sesuai kasus.
- Buat dokumentasi API menggunakan OpenAPI atau Postman.
